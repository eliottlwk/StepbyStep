# Step by Step — Prototype

Prototype frontend minimal (React + Vite) basé sur le composant fourni.

Quick start

```bash
# depuis le dossier du projet
npm install
npm run dev
```

Notes
- Le frontend tourne en local et stocke les tâches en mémoire (pour prototype rapide).
- Pour persistance, connecter à Supabase ou ajouter un backend; utiliser `db/001_init.sql` comme base de données relationnelle.
